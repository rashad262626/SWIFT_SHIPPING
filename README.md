# SWIFT_SHIPPING
منظومة Swift360 لإدارة الشحن هي نظام متكامل يهدف إلى تنظيم عمليات الشحن وإدارة العملاء والطلبات والخزائن المالية بكفاءة عالية. تتيح المنظومة إنشاء وتتبع الطلبات، إدارة كشوف حساب العملاء، متابعة خزينة التشغيل وصندوق الإيرادات، وإصدار تقارير يومية ومالية مفصلة مع إمكانية تصديرها بصيغة PDF، مما يساعد على اتخاذ قرارات دقيقة وسريعة.
